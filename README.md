# ActivityLifeCycle

### When application is Launched first time
```
onCreate
onStart
onResume
```
### When middle button is pressed
```
onPause
onStop
```
### When middle button is pressed and then activity is brough back to foreground again
```
onStart
onResume
```
### When middle button is pressed and then activity is brough back to foreground again
```
onStart
onResume
```
### WHen app is killed
```
onPause
onStop
onDestroy
```
Note: onDestroy is not called in some cases
